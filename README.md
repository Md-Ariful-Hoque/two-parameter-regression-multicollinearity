# Two-Parameter Regression Under Multicollinearity

## Overview
This repository contains code and results for a Monte Carlo simulation study of two-parameter estimators under multicollinearity.

## Contents
- `R/`: helper functions
- `scripts/`: main scripts to run simulations
- `data/`: input data if needed
- `results/`: output tables and figures

## Methods
- Monte Carlo simulation
- Type I error comparison
- Power comparison
- Correlation settings: 0.80, 0.90, 0.95, 0.99
- Sample sizes: 30, 50, 100, 200
- Predictors: 3, 5, 10
